在家里的日子这么短暂哟。

Posted on 2015-09-20 | | Visitors:

在家里就是不愿意做任何事情，包括之前想好的看php，写申请文书，一样都没有做到…在家唯一想做的事情就是睡觉：记得在一本小孩子的书里看到过，一个人如果走了很远很远的路，见了很多很多的人，最后累了伤心了，就到中国最北方的一个村子里，在村子里有一个土豆花花的房子，那些走了很久很久又累又伤心的人到了那里住下来，就开始睡觉，从冬天睡到夏天，一睡就是好几个月。等睡醒了，出来就变成一个白白胖胖的人，脸上又有了血色，之前经历的一些都像是一场梦一样，这个人走出来，就开始新的生活了。

我不是那个又伤心又累的人，心情也的确没有什么大起大落，正因为这样，才让我感觉到可怕，我的自传里没有变化是可怕的：今天下午去姑姑家，大姑，二姑，三姑，大姐，在那里打麻将，没有很高工作来源的人在社区做一份可有可无的工作，这是小城市里人们都觉得安逸的生活：平日清闲的工作，晚上看看电视，周末打打麻将，邻里之间串串门，过着省水省电省钱的生活，一周过去了，十年过去了。我看到大姐从我有记忆的十几岁便成了现在三十几岁，经营一家小药店的姐姐，我心里就想我的生活不应该要变成这样的，我从大一开始想啊想，我的人生是体验，人生未结束，体验未完成。甚至很消极的想，就算我去吸毒去打架，那也是体验，也是如果我打麻将就一辈子不会体验到的经历。现在想，就是一个有一个完成自己的目标，无论是学什么，还是要去做什么。可是连目标都不清晰啊！我正在从传统你问我答的教育中缓缓的挣扎着。

最近的我总是要说，什么都不想做呀。可是心里却在为想完成缺没有完成的事情干着急。这是为什么呢？没有很渴望，没有一个节奏来控制自己。大四生活就是这样，稍不注意就好几天过去了。我又是一个这思考那思考的人，思考来思考去就是很低落，为什么会低落呢？因为想做好的事情没有做。定下的目标没有做，也许是，下一刻就忘记了。这怎么可以呢，于是我要尝试一种趁早换新的方案了。

在家里的日子好短暂，什么都不想的日子，每天想的就是，今天吃什么，明天吃什么…无休止的睡觉，睡觉…可是明天我就要回去了，回到那个做着这件事想着下一件事的北京去，用好我在北京的最后的四个月，给自己一个对得起的答案吧。

挣扎的时候一定会过去。

Posted on 2015-05-24 | | Visitors:

想过，这段时间让我跳过去，不过了吧。早上起来搬着电脑到实验室，十点钟去上课，中午吃饭的时候洗头，然后开会，然后上课，实验室，晚上准备面试、大创项目、服务团队讲解、PSC… 上公交、下公交、上地铁、下地铁… 这些东西，你放弃了这些那些，你做出多少努力，最后还是没有任何结果的时候，应该如何去想呢。

那么就只有从过程里面去总结，这些经历的过程当中，真的是失败了就一定不值得么。

记录下来，这些东西，亲身经历之后才知道它是如何值得，如何心酸，如何让我知道路太长太长。只是知道我现在仍旧没有走出这些挣扎的时候。

这些挣扎的时候几乎没人可以说。我可以做到在工作中不表露自己感情，但是面对我喜欢的人们我不喜欢这样。面试被IBM搞错了之后，那两天一直就坐在床上哭，为啥呢？就是止不住啊，就想着，为啥是我，为啥是我，为啥是我… 努力这么久，为啥没有结果？就是一个人在战斗的感觉，慢慢败下阵来，慢慢怀疑自己…

心里特别不淡定，几乎无法做下去事情，晚上十一点半的时候我想我下去走走吧，真的不想坐在床上一直哭下去，买了一瓶脉动在校园里面走，听歌，想着过了十二点我明天一定还得继续下去，一定还得继续下去。

我想我该如何准备接下来的雅思，如何面对期末，每想到这些事情不想再想下去，可是是的还要继续下去，还要继续下去，还要继续下去。

一定还有很多很多的选择，失败之后不是绝路，现在就是要做什么呢，是让自己努力振奋起来，振奋起来。

没有结果的事情，总会有个结果的。

​ 双十一一年仅此一天的双十一。本来是想要在双十一的时候发出来，后来再回来写，已经是17号了。

​ 当时因为重感冒而睡了一个对时于是错过了朋友圈大家双十一的刷屏，淘宝又创下了38分钟100亿的成交量，吃早饭的时候是听说阿里众人在像看卫星发射一样的中央控制大厅来观看双十一的成交量。心里想啊原来是这样。大众的消费被直接旁观和看穿一般就像，大伙一群有一群正在工作的蚂蚁：“这边完成任务啦！” “那边攻下了好多亿的小沙粒！”

其实我可以自己吃早饭，又快，又不用叫别人起床，不用等待，早上有更多的时间做自己的事情，还可以在路上听歌，想想，自己吃饭很好啊。…后来sec小伙伴偶然倡议一起吃早饭，我说可以叫大伙，名曰morning-call….于是女孩子们提早25电话之，男孩子就提早20分钟电话之，前一天晚上吼一声约否，何时，众人拍板定下。我的闹钟响起来的时候是六点五十五分，然后叫女孩子，然后穿好衣服的时候是七点钟，然后叫男生们，爬下来洗漱。这个时候水房里没有人，热水是我的，水龙头也是我的。打一整盆温水来刷牙洗脸，惬意的事情。

瑞利法有时候会早到几分钟来背单词，我会看着手机准时到，玥儿和蛋会正好小跑着到，秀兰有时候会一起来，豆也是偶尔，从大电视到食堂慢慢走，秀兰来的时候就欣赏秀兰的大腿，秀兰是我爱的妹子，96年，长得水灵，我喜欢的女孩子的高高尖尖的鼻梁…惊叹的、每每admire suddenly的口语…喜欢的女孩子说话温柔细语，不似我一样扯着大嗓门吼叫…玥儿来的时候就欣赏林地的眼睛和嘴，第一次见玥儿姐姐的时候觉得这是一个女子，后来越见到越觉得美，晚上sec的时候我就盯着她的脸看，笑的时候美，故作生气的时候也美，哎呀真幸福。
